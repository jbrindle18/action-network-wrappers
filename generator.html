This tool themes an Action Network page wrapper — the frame (colours, fonts, logo, layout, footer and share buttons) that wraps around your AN action: a petition, form, sign-up or donation page. You change everything with the controls below — no code needed.

Want to see this template in use by the Romanian feminist organisation Centrul Filia? Click here.

How to use it

  1. Work through the sections below — set your colours, fonts, logo, layout and share options. The preview values update as you type.
  2. Click Copy header and paste it into the Header box of your AN page wrapper. Then click Copy footer and paste it into the Footer box. (In AN: Start Organizing → your group → Page wrappers → new/edit wrapper.)
  3. Save the wrapper in AN, and set your action to use it. Done — your action now wears your branding.

What each button does

  • Copy header / Copy footer — the two halves of the wrapper, ready to paste into AN's matching Header and Footer fields. This is the normal way to use the tool.
  • Save wrapper — saves your themed wrapper as a single wrapper.html file. That's your saved copy: keep it as a backup, hand it to someone else, or reopen it later to change the colours.
  • Import saved wrapper… — opens a wrapper.html you saved earlier and reads its colours, fonts, logo and share options back into the form, so you can pick up where you left off and edit.
